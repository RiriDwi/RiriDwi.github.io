GitHub Pages

Berikut adalah cara pembuatan GitHub Page dengan sistem Git

Langkah - langkah :

1. Membuat repository baru dengan nama (username.github.io)

2. Klik kanan pada folder yang akan kita upload ke GitHub dan pilih 'Git Bash Here'

3. Melakukan konfigurasi awal dengan 'git config'

4. Menginisialisasi folder menjadi repository dengan 'git init'

5. Gunakan 'git add .' untuk melacak semua perubahan pada file

6. Melakukan commit dengan menggunakan 'git commit â€“m' untuk menyimpan perubahan yang dilakukan.

7. Memindahkan branch yang tadinya Master menjad main dengan 'git branch -M main'

8. Menghubungkan komputer kita dengan GitHub dengan 'git remote add origin'

9. Mengupload hasil kodingan kita ke GitHub dengan 'git push'

10. Refresh halaman repository kita dan akan terlihat bahwa file telah berhasil diapload ke GitHub
